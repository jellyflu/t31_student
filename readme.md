 

# t31_student 
T31班学生管理系统,采用swing + JDBC开发的一个简单学生管理系统,实现了基本的CURD及姓名模糊查询的功能。

# 如何运行
java -jar t31_student.jar 或者javaw -jar t31_student.jar
